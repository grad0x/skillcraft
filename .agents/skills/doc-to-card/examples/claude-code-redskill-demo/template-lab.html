Recommendation / 模板建议

A. Editorial Tech / 技术杂志风

Best for: 技术深读、开发者向解释、需要专业感的首图。

Strength: 标题和对比结构强，来源边界显得可信。

Weakness: 收藏感稍弱，连续三张会偏冷。

First demo: 可用，但需要给 Cards 2-3 增加更多节奏变化。

B. Soft Knowledge / 小红书轻知识风

Best for: 小红书收藏卡、AI 工具学习者、轻解释入口。

Strength: 亲和、易扫读，角色母题能建立系列记忆点。

Weakness: 需要严格控制角色比例，避免显得幼稚。

First demo: 推荐作为第一版 RedSkill demo 的主方向。

C. Workflow Diagram / 工作流解释风

Best for: Agent workflow 概念解释、内部培训、流程型卡片。

Strength: 逻辑路径最清楚，能直接解释 General Chat 到 Workflow 的差异。

Weakness: 首图情绪弱，容易像培训 slide。

First demo: 可作为 Card 2 或 Card 3 的结构参考，不建议做首图主方向。

Recommended next step: choose one template direction, refine it into the master card template, then apply it to Cards 2 and 3.
