# Source Snapshot

本 demo 使用 A/B test 中的 controlled source snapshot / 受控来源快照。它不声称已阅读 Claude Code 完整官方文档。

## Controlled Source Snapshot / 受控来源快照

- Claude Code is an agentic coding tool.
- It lives in the terminal and can also be used in IDE or GitHub workflows.
- It helps developers work with codebases through natural language.
- It can execute routine coding tasks, explain complex code, and handle git workflows.
- It is designed for developer workflows rather than general chat.

## Evidence Boundary / 证据边界

可以说：

- Claude Code 是 agentic coding tool / 代理式编程工具。
- 它面向 developer workflows / 开发者工作流，而不是 general chat / 通用聊天。
- 它 lives in the terminal / 位于终端中，也可用于 IDE 或 GitHub workflows。
- 它帮助开发者通过 natural language / 自然语言处理 codebases / 代码库。
- 它可以执行 routine coding tasks、解释 complex code、处理 git workflows。

不能说：

- 价格、性能、安装方式、可用区域、模型、权限、用户数据。
- 支持的具体 IDE 名称或完整 GitHub 自动化范围。
- “一键替代程序员”或“完成所有开发任务”。
- 任何官方未在 source snapshot 中声明的结论。
