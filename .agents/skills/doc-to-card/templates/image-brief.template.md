# Image Brief Template

用于把 `doc-to-card` 的卡片草稿转成 image production brief / 图片生产简报和 image prompt / 生图提示词。本模板不生成图片文件。

## Series Image Brief / 系列图片简报

- Series title / 系列标题:
- Platform / 平台:
- Aspect ratio / 画幅比例:
- Visual style / 视觉风格:
- Color direction / 色彩方向:
- Typography direction / 字体方向:
- Motif setting / 视觉母题设置:
- Consistency rules / 一致性规则:
- Forbidden visual patterns / 禁用视觉模式:

## Cards / 单卡生产简报

### Card <number>: <card title>

- Content copy / 内容文案:
- Layout brief / 版式说明:
- Visual elements / 视觉元素:
- Image prompt / 生图提示词:
- Negative prompt / avoid:
- Text readability notes / 文字可读性提醒:

## Notes / 说明

- 如果需要精确文字，建议在 Canva/Figma/HTML/CSS 中排版文字，不依赖图像模型生成小字。
- 生图提示词只描述视觉风格、构图、插画和背景，不新增来源事实。
- 如果使用 mascot 或 character-pair，必须保持原创和系列一致性。
