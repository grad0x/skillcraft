# Style Profile Template

用于在生成卡片前明确 style profile / 风格配置。可以由用户填写，也可以由 Agent 基于渠道和目标读者推断，并把推断标为 assumption / 假设。

## Style Profile

- Source / 来源:
- Target audience / 目标读者:
- Channel / 渠道:
- Card count / 卡片数量:
- Density / 信息密度:
- Tone / 语气:
- Visual direction / 视觉方向:
- Title strategy / 标题策略:
- Evidence requirement / 证据要求:
- Technical depth / 技术深度:
- CTA style / 行动引导:
- Things to avoid / 禁用表达:
- Liked style examples / 喜欢的风格样例:
- Disliked style examples / 不喜欢的风格样例:

## Notes For Agent

- 如果用户给了风格样例，只抽象风格维度，不复制受版权保护的具体表达。
- 如果用户未指定某项，用渠道和目标读者推断，并在输出中标记为 assumption。
- 风格不能改变 source facts / 来源事实和 evidence boundaries / 证据边界。
